PARA EJECUTAR UN PROGRAMA DE JAVA CON PARÁMETROS:
1.
- IR A RUN
- CONFIGURACIÓN DE PROYECTO
- CUSTOMIZE
- RUN (SELECCIONAR CLASE PRINCIPAL, INSERTAR EL NOMBRE DE LOS PARÁMETROS Y LA RUTA DE LA CARPETA DONDE ESTA EL FICHERO)

2.
- CONTRUIR EL PROYECTO
- ABRIR UNA TERMINAL EN LA RUTA DEL PROYECTO
- EJECUTAR (JAVA -JAR NOMBREAPP.JAR ARGUMENTO)
